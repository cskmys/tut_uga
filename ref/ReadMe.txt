Type of files:
.udb files: project files to be opened in code browsing software called Understand/C++(It's 15 day trial license, you can write an email to them saying you are student and they'll give you free license )
.adoc files: asciidoc files that contains code walkthrough. They can be opened with AsciidocFx

Folders:
make: summary of all the make process(you should read this if you want to use your own code browsing software)
machine, threads: code walkthrough for corresponding folder in source directory
